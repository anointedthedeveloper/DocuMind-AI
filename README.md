# DocuMind AI

Scan, extract, and analyze text from documents using OCR and AI — right in your browser.

## Features

- **OCR Extraction** — Upload images or PDFs and extract text instantly
- **AI Analysis** — Automatic document categorization and summarization
- **Batch Scanning** — Process multiple documents at once
- **Export to PDF** — Save extracted text as a PDF
- **Document History** — Review and manage previously scanned documents

## Tech Stack

- React + TypeScript + Vite
- Tesseract.js (OCR engine)
- Tailwind CSS + shadcn/ui
- Supabase (AI analysis backend)

## Getting Started

```bash
npm install
npm run dev
```

## Usage

1. Drop an image or PDF onto the upload area
2. Click **Scan Document**
3. View extracted text and AI analysis
4. Export or manage documents from history
